User Manual

1. Index

On the main page we have a brief description of the application with the most important points. At the top is the search engine to find a specific user and as you type, suggestions appear. Then we have the explore button (“EXPLORE”) to see all the users available with the public account. To the right of this we have the SIGN IN and REGISTER buttons.

1.1. Register and Login

By clicking on the login and register buttons, the respective forms appear. In the forms where the password is entered, you can see what is being written by looking at it. When registering, if the data is correct, you are automatically logged in.

1.2. Explore

Here you can like if you are logged in, if not, you can only see people's profiles by clicking “view profile”

1.2.1 View Profile

Here the client can click on the links that the user has added.

2. Panel

First we have a new navigation bar when logging in to the user panel. In preview we can see a preview of the profile. In links you can: view, edit, delete and add the links that we want to be displayed on our profile. And finally in profile we can edit the missing information of the profile, such as the description, if we want the user to be shown in the explore tab, change the profile photo or the background photo, change the username that is going to be shown on the screen, change the password and change the style of the profile.

2.1. Profile

When logging in we will have the form to personalize the profile as the first screen.

2.1.1. Share Link

First we have the option to share our link or preview it to see what people would see. Then we have the form to upload the profile image, the background image, to be able to change the username that you want to show on the card, change the password, the color theme, the description and if you want the account to be shown in the explore or not.
